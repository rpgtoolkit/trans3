Help

This is the manual of the RPGToolkit, Version 3—the powerful role-playing game creation system. These pages contain all the information you need to begin making your game. Additional reading—help, information, articles, and tutorials—can be found on the constantly updated official site. All content is written for toolkit users by toolkit users.

Looking for something specific? Try searching.

If you're using this help system for the first time, choose the quick-start guide, or, if you're feeling adventurous, jump right in and have a play with the editor. If you experience any problems or think you might have found a bug, first check the technical issues and frequently asked questions; if you can't find what you're looking for, there are literally hundreds of users who are willing help online at the official forums. We hope you find this manual helpful and concise.
